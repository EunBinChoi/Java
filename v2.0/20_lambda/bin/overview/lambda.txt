[ 람다식 (lambda expression) ]
- 자바 8부터 사용
- 객체지향 언어보다는 함수지향 언어 가까움
- 익명함수를 생성하기 위한 식 (일회성 함수)

- 람다식 기본 문법
(매개변수, ....) -> {실행문; .......}
ex) (a) -> {sysout(a);} 


※ 프로그래밍 패러다임
- 명령형 프로그래밍 : 어떻게 ? (HOW)
ex) 절차지향 프로그래밍 (C, C++), 객체지향 프로그래밍 (JAVA, C++, C#)

- 선언형 프로그래밍 (함수형 프로그래밍) : 결과적으로 무엇을 할 건데? (WHAT)
ex) 람다식 

int x;
int y;
f1 = (x, y) -> return x + y;
f1.method(x, y); // add
f1 = (x, y) -> sysout(x + " " + y);
f1.method(x, y); // print
